Design-pattern references (NO code copied)
--------------------------------------------------------------------------------
The v1.1 interface adapts interaction patterns studied in:
  * Subtitle Edit - https://github.com/SubtitleEdit/subtitleedit (MIT):
    segment grid + separate active-segment editor, strong selected-row
    highlighting, video/grid selection sync, keyboard-first editing.
  * VideoTranscriber - https://github.com/DataAnts-AI/VideoTranscriber (MIT):
    categorized collapsible settings, stage-by-stage progress with elapsed
    times, local Ollama model discovery, result tabs.
All LecturePack UI code is an original PySide6 implementation of these
concepts; no source code was copied or transplanted from either project.

Ollama (NOT bundled)
--------------------------------------------------------------------------------
LecturePack can optionally talk to a locally installed Ollama server
(https://ollama.com, MIT License) over http://localhost:11434. Neither the
Ollama runtime nor any language model is bundled with LecturePack; AI
features stay disabled when Ollama is absent.
